Automacao do Carrinho

1

Instale uma extensao de userscript no Chrome, como Tampermonkey ou Violentmonkey, pela Chrome Web Store.

2

Abra o painel da extensao, crie um novo script ou edite o script existente, apague todo o conteudo e cole a versao abaixo.

3

Salve o script. Depois volte ao QueroPedirMarmitas e use o botao Montar Carrinho no Site. Ao funcionar, uma caixa de status aparece no site da Marmitas da Lulu antes de abrir o carrinho.

Abrir Chrome Web Store Ver Arquivo Cru

O erro do Chrome aparece porque o navegador bloqueia a instalacao direta de apps, extensoes e scripts a partir de sites locais como localhost. Instalando pelo painel do Tampermonkey, o script fica no seu navegador e roda somente no site da Marmitas da Lulu. Se a caixa de status nao aparecer no site, confirme no Tampermonkey se o script esta ativo e se o cabecalho mostra `@version 1.2.0`.
